# Словарь словосочетаний - Задание 3

Приложение для обработки текстов на английском языке и создания словаря словосочетаний с удобным графическим интерфейсом.

## Возможности

- **Загрузка текстов** из файлов различных форматов (TXT, RTF, PDF, DOC, DOCX)
- **Обработка текста** с извлечением слов (лемм) и их частей речи
- **Извлечение словосочетаний** с подсчетом частоты встречаемости
- **Управление словарем**: просмотр, фильтрация, редактирование, удаление
- **Синтез словосочетаний** — генерация новых фраз из случайных слов
- **Тестирование производительности** — замеры скорости операций
- **Импорт/экспорт** словаря в форматах TXT и CSV

## Требования

- Python 3.10+
- PyQt6
- spaCy (с моделью en_core_web_sm)
- SQLAlchemy
- Другие зависимости из requirements.txt

## Установка

1. Клонируйте репозиторий и перейдите в папку проекта

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Загрузите модель spaCy:
```bash
python -m spacy download en_core_web_sm
```

## Запуск

```bash
python run.py
```

или

```bash
python -m word_collocation_dict.main
```

## Руководство пользователя

### Важное примечание

**База данных очищается при каждом запуске приложения.** Для сохранения результатов используйте функцию экспорта.

### Вкладка «Загрузка файла»

1. **Выбор файла**: Нажмите «Открыть файл» в панели инструментов или «Обзор...» на вкладке
2. **Загрузка**: Нажмите «Загрузить файл» — текст появится в поле
3. **Редактирование**: При необходимости отредактируйте текст в поле
4. **Обработка**: Нажмите «Обработать текст» внизу вкладки — слова и словосочетания будут добавлены в словарь

### Вкладка «Словарь»

Просмотр и управление извлеченными словами:

**Фильтры:**
- **Поиск** — фильтрация по началу слова (работает в реальном времени)
- **Часть речи** — фильтр по грамматической категории (NOUN, VERB, ADJ, ADV, PROPN)
- **Кол-во сочетаний** — фильтр по точному количеству словосочетаний у слова

**Действия со словами:**
- **Добавить слово** — открывает диалог для ручного добавления леммы:
  - Введите слово (лемму)
  - Выберите часть речи из списка: NOUN, VERB, ADJ, ADV, PROPN
  - Проверка на дубликаты (нельзя добавить существующее слово)
- Выберите слово для просмотра его словосочетаний
- Двойной клик по ячейке позволяет редактировать лемму или часть речи
- **Удалить слово** — удаляет выбранное слово и все его словосочетания

**Действия со словосочетаниями:**
- **Добавить словосочетание** — открывает диалог для добавления нового сочетания:
  - Введите текст словосочетания (например: "buy a book")
  - Если словосочетание уже существует — частота увеличивается на 1
  - Система автоматически определяет значимые слова (существительные, глаголы, прилагательные, наречия) через spaCy
  - Служебные слова (артикли "a/the", предлоги и т.д.) автоматически фильтруются
  - Для каждого нового слова открывается диалог выбора части речи (можно подтвердить предложенную spaCy или выбрать другую)
  - Можно пропустить добавление конкретного слова кнопкой "Пропустить"
- **Удалить словосочетание** — удаляет выбранное словосочетание

**Удаление слов:**
- При удалении слова все связанные с ним словосочетания также удаляются

### Вкладка «Синтез»

Генерация новых словосочетаний:

1. Загрузите слова в морфо-словарь:
   - «Загрузить из текущего текста» — из текста на вкладке «Загрузка файла»
   - «Загрузить из словаря» — из слов в базе данных
2. Нажмите «Синтезировать словосочетание»
3. Результат появится в поле ниже

### Вкладка «Тестирование»

Запуск тестов производительности:
- **Тест обработки текста** — замеры на текстах разной длины
- **Тест поиска** — скорость поиска слов и получения словосочетаний
- **Тест фильтрации** — скорость фильтрации по части речи и тексту

Результаты отображаются с точностью до миллисекунд.

### Файловые операции (меню «Файл»)

Доступны в панели инструментов через кнопку «Файл ▼»:

- **Импорт словаря** — загружает словарь из файла (заменяет текущий)
- **Добавить к словарю** — добавляет словарь из файла к текущему
- **Экспорт словаря** — сохраняет текущий словарь в файл (TXT, CSV)
- **Очистить словарь** — удаляет все слова и словосочетания

### Вкладка «Терминология»

Справочник лингвистических терминов:
- Лемма, лексема, словоформа
- Части речи: NOUN, VERB, ADJ, ADV, PROPN
- Словосочетание, частота, синтез

### Вкладка «Помощь»

Памятка по работе с программой.

## Архитектура

Приложение построено по принципу разделения ответственности:

- **Миксины** (`ui/mixins/`) — каждый отвечает за свою функциональность:
  - `FileTabMixin` — загрузка и обработка файлов
  - `DictionaryViewMixin` — просмотр, фильтрация и отображение словаря
  - `DictionaryEditMixin` — добавление и удаление слов/словосочетаний
  - `DictionaryTabMixin` — комбинированный миксин (наследует View и Edit)
  - `SynthesizerTabMixin` — синтез словосочетаний
  - `StorageMixin` — импорт/экспорт/очистка

- **PerformanceTester** (`benchmark/`) — отдельный класс для тестирования

- **MVC-подобная структура**:
  - Model: `database/models.py`
  - View: `ui/`
  - Controller: логика в миксинах и `main_window.py`

## Особенности реализации

- **Многопоточность** — обработка текста выполняется в отдельном потоке (QThread) для предотвращения зависания UI
- **Динамическая фильтрация** — фильтры в словаре применяются в реальном времени
- **Модульная архитектура** — легко добавлять новые форматы файлов и функции
