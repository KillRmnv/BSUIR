# BSUIR Academic Projects Repository

This repository contains academic projects and laboratory works from various courses at BSUIR (Belarusian State University of Informatics and Radioelectronics). Each directory represents a different subject or course, with detailed README files explaining the specific implementations.

## Directory Structure

- **AIPOS** - Hardware and Software of Networks
  - Learning materials for network subjects and proxy server implementation in Python with GET request caching

- **AOIS** - Hardware Support of Intelligent Systems
  - Laboratory works for low-level implementation of computer systems
  - Contains 7 independent Maven projects (AOIS1 through AOIS7) covering:
    - Number representation and binary arithmetic
    - Logical expression analysis and normalization
    - Logical function minimization (calculation, tabular, and Karnaugh map methods)
    - Tetrad converter and logical circuit synthesis
    - Digital vending machine (finite state machine)
    - Hash table modeling with collision resolution
    - Associative processor and memory systems

- **EYAZIS** - Word Collocation Dictionary
  - Application for processing English texts and creating a dictionary of word collocations with GUI
  - Features: text loading, linguistic processing, collocation extraction, dictionary management, synthesis, testing, import/export
  - Built with Python 3.10+, PyQt6, spaCy, SQLAlchemy


- **MOIS** - Mathematical Foundations of Intelligent Systems
  - Source files for formalization of Semantic code concepts (ScS and ScG) as part of OSTIS technology

- **OIIS** - Fundamentals of Image and Signal Processing
  - Laboratory works covering:
    - Fast Fourier Transform implementation and visualization
    - Image filtering using PIL/Pillow
    - Brightness adjustment with OpenCV
    - Image segmentation (edge-based and region-based methods)
    - Stereo image processing
    - Additional materials on red-black trees and mapping

- **OS** - Operating Systems Laboratory Works
  - Five laboratory works covering:
    - File search and filtering (bash script)
    - Process management (C++ with fork/exec)
    - Multithreading and synchronization (producer-consumer with semaphores)
    - Custom memory manager implementation
    - File system simulation with FAT concepts

- **OTIS** - Graph Theory Visualization Tool
  - Qt-based application for creating, manipulating, and analyzing graphs
  - Features: interactive graph creation, graph algorithms (Eulerian cycles, tree identification, shortest path), visual feedback, multi-graph support

- **PBZ** - Academic Projects
  - Three distinct projects:
    1. Database Design Project (supplier-parts-projects schema in SQL)
    2. Library Management System (Java/Maven with PostgreSQL stored procedures)
    3. Ontology Development Project (OWL/RDF with SPARQL queries and web application)

- **PPOIS** - Object-Oriented Programming in Software Systems
  - Educational materials, lab works, and course projects including:
    - Lecture materials on OOP concepts
    - Laboratory works: Set implementation, Document management, Airport simulation, Graph implementation
    - Course projects: Law Registry System (Java), Customer Processing System (JavaFX)

- **SIMZIS** - Information Security Laboratory Works
  - Laboratory works focusing on cryptography and information security:
    - Password generator and brute force analysis
    - Table transposition cipher
    - Diffie-Hellman key exchange
    - RSA cryptosystem with digital signatures
    - Network firewall technology
    - OpenSSL cryptographic library
    - Network traffic analysis with Snort

- **StatOIV** - Statistical Analysis and Data Processing
  - Python scripts for statistical analysis:
    - Weather classification using multiple ML algorithms
    - Weather clustering analysis
    - Weather regression analysis
    - Student habits and performance analysis

## Common Technologies

Various projects utilize different technology stacks including:
- Languages: Python, Java, C++, JavaFX
- Frameworks/Libraries: PyQt6, spaCy, SQLAlchemy, NumPy, Pandas, Scikit-learn, OpenCV, PIL/Pillow, Qt, Maven, JUnit 5, Lombok
- Databases: PostgreSQL, SQLAlchemy ORM
- Web Technologies: OWL/RDF, SPARQL, Java web applications
- Build Systems: Maven, Make/QMake

## Running Projects

Each directory contains its own README.md with specific instructions for building, running, and using the projects. Please refer to the individual README files in each directory for detailed setup and usage instructions.

## Notes

All projects are developed for educational purposes as part of BSUIR coursework. They demonstrate practical implementation of concepts taught in the respective courses.
